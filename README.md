# EBONLINE! (Pretty)
All Mother/Earthbound Games emulated online using NeptunJS.


## Page Nomenclature
* Earth0 = Mother 1/EarthBound Zero
* Earth1 = Mother 2/EarthBound
* Earth2 = Mother 3/EarthBound 2
* Earth3 = EarthBound Halloween Hack (by Toby Fox)
